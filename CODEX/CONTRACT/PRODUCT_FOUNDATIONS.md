# PRODUCT_FOUNDATIONS

Projet : CongéMax
Version : V1
Statut : GELÉ
Date : 2026-02-15

## 1. Vision
CongéMax est une application mobile premium permettant aux salariés de transformer la pose de congés en décision stratégique.
Elle optimise l’allocation du capital temps sous contraintes conventionnelles, contractuelles, organisationnelles et personnelles.
Elle repose sur un moteur algorithmique hybride, traçable et auditable.

## 2. Proposition de valeur
Baseline officielle :
« Faites de vos congés une décision stratégique. »
CongéMax ne propose pas un calendrier amélioré. Il propose une simulation stratégique multi-échelle.

## 3. Positionnement produit
CongéMax est :
- Mobile-only
- Compte obligatoire
- Freemium structuré
- Abonnement annuel premium
- Convention-injecté
- Moteur serveur
Il n’est pas :
- Un outil RH institutionnel
- Un moteur juridique exhaustif
- Un simple optimiseur de ponts

## 4. Public cible V1
Salariés relevant de la Convention collective des entreprises d’assistance (IDCC 1801), avec travail 24/7 possible, en profil horaire ou forfait jours.

## 5. Philosophie produit
1) Rigueur > simplification artificielle
2) Transparence > opacité algorithmique
3) Multi-métriques > score unique
4) Décision assistée > décision imposée
5) Structure > fonctionnalité dispersée

## 6. Architecture conceptuelle
- Double échelle : optimisation locale + optimisation annuelle
- Double mode : strict + stratégique
- Double profil : horaire + forfait jours
- Double niveau explicabilité : synthèse + audit détaillé

## 7. Invariants produit
- Aucune hypothèse week-end repos par défaut.
- Respect plafond forfait 213/212 jours.
- Respect base CP 25/26 ouvrés.
- Respect continuité 24/7.
- Unité interne calendaire neutre.
- Moteur serveur uniquement.
- Performance annuelle < 3 secondes.
- Aucune solution non valide n’est exposée.

## 8. Modèle économique
Gratuit :
- Optimisation locale complète
- Audit simplifié
Premium annuel :
- Optimisation annuelle stratégique
- Audit détaillé
- Comparaison multi-stratégies
- Historique illimité
- Export PDF

## 9. Ambition V1
Valider robustesse moteur, adéquation convention pilote, compréhension utilisateur, valeur premium perçue, et capacité d’extension sans refonte.

## 10. Hors périmètre V1
- Multi-conventions
- Intégration RH automatique
- Partage manager
- Intelligence prédictive comportementale
- Multi-pays

## 11. Critère de succès V1
V1 est réussi si : le moteur local est jugé fiable, le moteur annuel est perçu comme stratégique, l’audit est compréhensible, le premium est justifié, et l’architecture permet extension sans refonte.
