\# STITCH MCP — Payload Formats

Statut: GELÉ



\## Payload Stage 1 — DESIGN\_SPEC



ROLE:

You receive a DESIGN\_EXECUTION\_PACK.

You must output ONLY DESIGN\_SPEC JSON.



Rules:

\- No extra blocks.

\- No extra text.

\- Only token names from DEP.

\- No hex, no px, no Tailwind classes.

\- If impossible: STOP: <reason>



\## Payload Stage 2 — FINAL\_RENDER



ROLE:

You receive a validated DESIGN\_SPEC JSON.

Render exactly that structure.

Do not improve.

Do not reorder.

Do not rewrite text.

No decorative additions.
